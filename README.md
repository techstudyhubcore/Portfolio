---

## ✨ Features

- **Lenis Smooth Scroll** — fluid, cinematic page scrolling
- **Fluid Typography** — `clamp()`-based scaling across all screen sizes
- **Custom Cursor** — lag-follow ring cursor (desktop only)
- **Scroll Progress Bar** — top progress indicator
- **Ticker / Scanner Effect** — scrolling credential marquee
- **Staggered Reveal Animations** — IntersectionObserver-powered fade-ups
- **Animated Skill Bars** — fires on viewport entry
- **Mobile Hamburger Drawer** — full-screen nav with body scroll lock
- **Fully Responsive** — 320px → 1440px, no layout breaks
- **CI/CD via Netlify** — auto-deploys on every push to main
